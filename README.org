* How to install
1. Install GNU stow.
2. Clone this repo.
3. Run the following commands in your terminal:
   #+begin_src sh 
   $ cd dotfiles
   $ stow .
   #+end_src

* Cleaning up symbolic links
1. Install GNU stow.
2. Run the following commands in your terminal:
   #+begin_src sh 
   $ cd dotfiles
   $ stow -D .
   #+end_src
