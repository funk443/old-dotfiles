* Installing
Either loading the Lisp file with SBCL or just execute the binary will
do.  Please *DO NOT* execute the script with root privilege, the script
will ask you for the password when needed.
#+begin_example
sbcl --load install.lisp
#+end_example

#+begin_example
./install
#+end_example
